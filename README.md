## ✨ Особенности

- 🔑 **Генерация шифра по числовому ключу** — каждый ключ создает уникальный шифр
- 📱 **Детерминированность** — одинаковый шифр на всех компьютерах
- 📏 **Настройка длины кода** — от 8 до 64 символов
- ⚡ **Настройка сложности** — 3 уровня защиты
- 🔒 **Защита паролем** — дополнительное XOR шифрование
- 💾 **Экспорт/импорт шифров** — сохранение в JSON
- 📊 **Статистика и анализ** — подробный разбор текста
- 🎨 **Современный GUI** — темная тема с анимациями
- 💻 **Командная строка** — для продвинутых пользователей
